1. Вивести статистику: загальна кількість всіх книг, їх вартість, їх середню вартість, мінімальну і максимальну ціну : 
	SELECT COUNT(Назва) AS `Кількість книг`, SUM(Цiна) AS `Загальна вартість`, AVG(Цiна) AS `Середня вартість`, MIN(Цiна) AS `Мінімальна ціна`, MAX(Цiна) AS `Максимальна ціна` FROM `table_1`
	
2. Вивести загальну кількість всіх книг без урахування книг з непроставленою ціною : 
	SELECT COUNT(Назва) FROM `table_1` WHERE Цiна IS NOT NULL

3. Вивести статистику (див. 1) для книг новинка / не новинка : 
	SELECT Новинка, COUNT(Назва) AS `Кількість книг`, SUM(Цiна) AS `Загальна вартість`, AVG(Цiна) AS `Середня вартість`, MIN(Цiна) AS `Мінімальна ціна`, MAX(Цiна) AS `Максимальна ціна` FROM `table_1` GROUP BY Новинка

4. Вивести статистику (див. 1) для книг за кожним роком видання : 
	SELECT YEAR(`Дата`) as `Рiк видання`, COUNT(Назва) AS `Кількість книг`, SUM(Цiна) AS `Загальна вартість`, AVG(Цiна) AS `Середня вартість`, MIN(Цiна) AS `Мінімальна ціна`, MAX(Цiна) AS `Максимальна ціна` FROM `table_1` GROUP BY YEAR(`Дата`) 

5. Змінити п.4, виключивши з статистики книги з ціною від 10 до 20 : 
	SELECT YEAR(`Дата`) as `Рiк видання`, COUNT(Назва) AS `Кількість книг`, SUM(Цiна) AS `Загальна вартість`, AVG(Цiна) AS `Середня вартість`, MIN(Цiна) AS `Мінімальна ціна`, MAX(Цiна) AS `Максимальна ціна` FROM `table_1` WHERE Цiна NOT BETWEEN 10 AND 20 GROUP BY YEAR(`Дата`)
	
6. Змінити п.4. Відсортувати статистику по спадаючій кількості : 
	SELECT YEAR(`Дата`) as `Рiк видання`, COUNT(Назва) AS `Кількість книг`, SUM(Цiна) AS `Загальна вартість`, AVG(Цiна) AS `Середня вартість`, MIN(Цiна) AS `Мінімальна ціна`, MAX(Цiна) AS `Максимальна ціна` FROM `table_1` GROUP BY YEAR(`Дата`) ORDER BY COUNT(Назва) DESC

7. Вивести загальну кількість кодів книг і кодів книг що не повторюються : 
	SELECT COUNT(Код) AS `Загальна кiлькiсть кодiв`, COUNT(DISTINCT Код) AS `Кiлькiсть унiкальних кодiв` FROM `table_1`

8. Вивести статистику: загальна кількість і вартість книг по першій букві її назви : 
	SELECT LEFT(Назва,1) AS `Перша літера назви`, COUNT(Назва) AS `Кількість книг (з літерою)`, SUM(Цiна) AS `Загальна вартість` FROM `table_1` GROUP BY LEFT(Назва,1)

9. Змінити п. 8, виключивши з статистики назви що починаються з англ. букви або з цифри : 
	SELECT LEFT(Назва,1) AS `Перша літера назви`, COUNT(Назва) AS `Кількість книг (з літерою)`, SUM(Цiна) AS `Загальна вартість` FROM `table_1` WHERE NOT LEFT(Назва, 1) REGEXP '[A-Za-z0-9]' GROUP BY LEFT(Назва,1)

10. Змінити п. 9 так щоб до складу статистики потрапили дані з роками більшими за 2000 : 
	SELECT LEFT(Назва,1) AS `Перша літера назви`, COUNT(Назва) AS `Кількість книг (з літерою)`, SUM(Цiна) AS `Загальна вартість` FROM `table_1` WHERE NOT LEFT(Назва, 1) REGEXP '[A-Za-z0-9]' AND YEAR(Дата)>2000 GROUP BY LEFT(Назва,1)

11. Змінити п. 10. Відсортувати статистику по спадаючій перших букв назви : 
	SELECT LEFT(Назва,1) AS `Перша літера назви`, COUNT(Назва) AS `Кількість книг (з літерою)`, SUM(Цiна) AS `Загальна вартість` FROM `table_1` WHERE NOT LEFT(Назва, 1) REGEXP '[A-Za-z0-9]' AND YEAR(Дата)>2000 GROUP BY LEFT(Назва,1) DESC

15.	Вивести статистику для книг новинка / не новинка: загальна ціна, загальна ціна в грн. / Євро / руб. Колонкам запиту дати назви за змістом :
